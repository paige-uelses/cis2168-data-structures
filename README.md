#CIS2168 Data Structures (Temple University)

Java implementations of data structures exercises:

Java implementations of core data structure exercises:
- **ZigZag.java** – Traverses a 2D array in zigzag order
- **Reverser.java** – Reverses arrays using two-pointer technique
- **Intersect.java** – Finds intersection of two sorted ArrayLists
- **Squash.java** – Removes consecutive duplicates from arrays
